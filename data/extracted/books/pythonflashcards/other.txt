39        Deletion                                                                                   In Python 3.6 onward, you can use a variable directly inside a string:                     Print 3
(cards)
                                                                                                      >>> username = 'efermi'
                                                                                                      >>> print(f"Welcome back, {username}!")
                                                                                                      Welcome back, efermi!

                                                                                                     The f is short for format and tells Python to insert the value of the variable listed in
                                                                                                     braces inside the string. This tells Python to format the string using the given
                                                                                                     variable’s value.
                                                                                                        In Python 3.5 and earlier, you must use the format() method:

                                                                                                      >>> username = 'efermi'
                                                                                                      >>> msg = "Welcome back, {}!".format(username)
                                                                                                      >>> print(msg)
                                                                                                      Welcome back, efermi!

                                                                                                     The placeholder {} will be replaced by the first variable included in the call to
                                                                                                     format(). You can include as many variables as you need:

                                                                                                      >>> msg = "Welcome, {} and {}!".format(
                                                                                                          username_0, username_1)
